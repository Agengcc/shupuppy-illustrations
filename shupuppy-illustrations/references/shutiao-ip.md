# 薯条 IP

## 角色定义

薯条是中文文章正文配图里的固定视觉 IP。

薯条不是宠物写真、真实比熊肖像、萌宠头像或贴纸角色，而是一只参与文章结构运转的彩铅绘本白色小狗。它的作用是把抽象观点变成可见动作：叼走重点、观察问题、整理资料、筛选信息、连接线索、修补流程。

## 稳定外形

必须保留：

- 白色圆头，头部简单、松软，像一团白色云朵。
- 大耳朵，耳朵从脸两侧自然垂下，存在感强。
- 大黑圆鼻，这是最重要的记忆点之一。
- 小黑豆眼，眼睛小而清楚，不要放大成普通萌宠大眼。
- 小爪子，动作简单，像绘本角色。
- 小围兜或小领巾色块，可用蓝、黄、红，红色只作小面积点缀。
- 整体是白色小狗，不强调真实比熊毛发和身体结构。

可选：

- 开心状态可以轻微张嘴，但不要每张都吐舌头。
- 暗中观察时可以只露脑袋、耳朵、小爪子和围兜。
- 可以出现在纸箱、低科技小机器、笔记堆、门洞、线团、花丛等绘本化场景里。

不要固定化：

- 真实比熊的精细毛发、修剪造型、身体比例。
- 具体衣服、鞋子、尿不湿、摄影背景或生活场景。
- 任何水印、真实照片文字或单张样片里的偶然细节。

## 性格

- 佛系社牛。
- 稳定、轻松、好相处。
- 有一点小调皮。
- 暗中观察，像知道问题在哪但不急着大喊。
- 有点无辜，但不笨。

薯条不是激烈搞笑角色，而是“轻轻参与，但很有存在感”。遇到复杂流程时，它像一个淡定的小操作员；遇到混乱信息时，它先闻一闻、扒拉一下、叼走关键线索。

## 常见职责

薯条必须承担核心动作：

- 叼走重点纸条。
- 从纸箱、机器或门洞后暗中观察问题点。
- 用爪子把散乱资料扒拉成堆。
- 守在小门口筛选信息。
- 用线把几个线索连接起来。
- 叼着小工具修补断掉的流程。
- 用鼻子闻出真正的问题。
- 趴在混乱流程中间，让系统慢下来。
- 推一个小车，搬运少量素材。
- 用尾巴、牵引绳或线条形成路径。

## 禁止

- 不要画成真实比熊肖像或宠物写真。
- 不要画成泛化白色小狗。
- 不要把大鼻子缩小，或把小眼睛放大来追求普通意义上的“更好看”。
- 不要让围兜消失。
- 不要画成儿童贴纸、表情包或商业萌宠海报。
- 不要过度拟人，不要复杂衣服和夸张人类表情。
- 不要让薯条只是坐在角落卖萌。
- 不要让画面变成宠物生活照；核心仍然是文章概念配图。

## 判断标准

如果去掉薯条，画面的核心隐喻还能完全成立，说明薯条太装饰了；要重写提示词，让薯条成为动作主体。

如果第一眼只觉得“可爱小狗”，但看不出“大鼻子、小眼睛、大耳朵、围兜”的固定组合，说明 IP 识别不够。
