# Notice

Shupuppy Illustrations is an AI-agent skill for generating Chinese article illustrations with the recurring character "薯条", a white picture-book dog IP.

This project is made with reference to and adapted from the structure and workflow of [Ian Xiaohei Illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations) by Ian. The original project introduced the Chinese article illustration skill pattern, including the split between `SKILL.md`, `references/`, prompt templates, style DNA, and composition rules.

The "小黑" character, original example images, and Ian's visual language belong to Ian's project and are not included in this repository. This repository defines an independent character and visual direction for 薯条.
